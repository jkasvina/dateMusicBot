Создаю нового бота в тг @BotFather.
Получаю токен своего бота "5026626038:AAGlVDusn4QPe6jGW4R41DQaz4J0qUDq1cs".
В настройках webstorm ищу autosave и устанавливаю его на каждые 15 секунд.

Создаю пустой проект в webstorm.
Затем запускаю команду npm init -y (создаётся package.json).
В корне проекта создаю index.js, в котором созаю переменную token.
Запускаю установку библиотеки-> npm i node-telegram-bot-api nodemon
(nodemon автоматически перезагружает сервер при каждом изменении в коде).
Созаём переменную для импорта библиотеки тг ботов.

Затем в package.json проверяем, что зависимости установлены и создаём скрипты для тестирования и запуска приложения.

В index.js создаю прослушку события message. Затем запускаю бот-> npm run dev.
Теперь если в боте отправить сообщение, то объект с информацией о msg отобразится в консоли.
